SISTEM PENDUKUNG KEPUTUSAN REKOMENDASI SATUAN PENGAMAN TERBAIK MENGGUNAKAN METODE SIMPLE ADDITIVE WEIGHTING 
Aplikasi ini dirancang untuk membantu proses pemilihan satuan 
pengaman terbaik berdasarkan hasil penilaian kinerja security secara 
objektif dan terstruktur. 
Aplikasi berbasis web ini di bangun 
menggunakan : - PHP Native - My Sql - HTML, CSS, Java Script 
Aplikasi ini memiliki Fitur : 
- Login
- Input Kriteria dan Alternatif
- Perhitungan Metode SAW Otomatis 
Cara Instalasi :
- Copy Folder source codenya (skripsweet) ke dalam folder htdocs
- Buka Aplikasi XAMPP Control Panel
- Aktifkan Module “ Apache dan MySQL “
- Klik Admin Pada Action MYSQL
- Setelah di klik otomatis akan Masuk ke dalam PHP MY ADMIN
- Buat database kosong  (saw_security)
- Import database (saw_security) yang ada pada file source code ke 
database kosong yang baru di buat My Sql - Tulis alamatnya pada browser ( Localhost/skripsweet/ ) untuk 
masuk pada Halaman Login pada aplikasinya
